#Noderplate

Noderplate is a simple node.js web app boilerplate.

##Technologies
* Express
* Coffeescript
* Stylus
* Eco
* Stitch
* Underscore
* Backbone
* Jquery

##Usage
    $ coffee ./app/server.coffee
